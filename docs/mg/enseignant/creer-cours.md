# Mamorona lesona

Torolalana ho an'ny mpampianatra te-hamorona sy hamoaka lesona ao amin'ny CNTEMAD LMS.

## Zavatra ilaina

- Kaonty mpampianatra voamarina
- Fidirana amin'ny module famoronana lesona

## Firafitry ny lesona

Ny lesona CNTEMAD dia manaraka ity firafitra ity:

```
Lesona
├── Fampahalalana ankapobeny
│   ├── Lohateny
│   ├── Famaritana
│   ├── Sampana
│   └── Ambaratonga
├── EC 1 (Élément Constitutif)
│   ├── Fampianarana 1
│   ├── Fampianarana 2
│   └── Quiz EC
├── EC 2
│   └── ...
└── Fanadinana farany
```

## Mamorona lesona vaovao

### 1. Fidirana amin'ny famoronana

1. Midira amin'ny maha-mpampianatra
2. Mandehana amin'ny **Ny Lesoko** > **Mamorona lesona**
3. Fenoy ny fampahalalana fototra

### 2. Fampahalalana ankapobeny

| Saha | Famaritana | Ohatra |
|------|------------|--------|
| **Lohateny** | Anaran'ny lesona | "Fampidirana Python" |
| **Kaody** | Famantarana tokana | "INFO-L1-PY01" |
| **Famaritana** | Famintinana ny lesona | "Ianaro ny fototra..." |
| **Sampana** | Departemanta | Informatika |
| **Ambaratonga** | L1, L2, L3, M1, M2 | L1 |
| **Sary** | Vignette ny lesona | 800x450px voatolotra |

### 3. Manampy EC

Tsindrio **+ Manampy EC** isaky ny élément constitutif:

```
EC: Fototry ny Python
├── Tanjona: "Amin'ny faran'ity EC ity, ny mpianatra dia hahay..."
├── Faharetana vinavina: 10 ora
├── Crédits: 3 ECTS
└── Filaharana: 1
```

## Mamorona fampianarana

### Karazana votoaty

#### Lahatsoratra manankarena

Ny éditeur dia manohana:
- **Matevina**, *mikirakira*, ~~voakosoka~~
- Lisitra
- Tabilao
- Bloc code misy loko
- Sary sy rohy
- Formule matematika (LaTeX)

```python
# Ohatra code aseho ao amin'ny fampianarana
def miarahaba():
    print("Miarahaba, CNTEMAD!")
```

#### Video

1. Tsindrio **Manampy video**
2. Alefaso ny rakitra (MP4 voatolotra)
3. Na apetaho rohy YouTube/Vimeo

::: tip Torohevitra video
- Faharetana tsara: 5-15 minitra isaky ny video
- Kalitao: 720p minimum
- Mitenena mazava sy antonony
:::

#### Tahirin-kevitra PDF

1. Tsindrio **Manampy tahirin-kevitra**
2. Alefaso ny PDF-nao
3. Ampio famaritana

## Mamorona Quiz

### Firafitry ny quiz

| Paramètre | Famaritana | Voatolotra |
|-----------|------------|------------|
| **Faharetana** | Fotoana voafetra | 20-30 min isaky ny EC |
| **Fanandramana** | Isan'ny fikasana | 2-3 |
| **Naoty fidirana** | Minimum hanekena | 60% |
| **Filaharan-kisendrasendra** | Mampifangaro fanontaniana | Eny |

### Karazana fanontaniana

#### QCM (safidy tokana)

```
Fanontaniana: Inona ny renivohitr'i Madagasikara?
○ Tananarive
○ Toamasina
○ Antananarivo ✓
○ Fianarantsoa

Feedback: Antananarivo, antsoina koa hoe Tana, no renivohitra.
```

#### QCM (safidy maromaro)

```
Fanontaniana: Inona avy no fiteny fandaharana?
☑ Python
☑ Java
☐ HTML
☑ JavaScript

Fanamarihana: HTML dia fiteny balisage, tsy fandaharana.
```

#### Marina/Diso

```
Fanontaniana: Python dia fiteny compilé.
○ Marina
○ Diso ✓

Feedback: Python dia interprété, tsy compilé.
```

## Mamoaka ny lesona

### Fanamarinana alohan'ny famoahana

- [ ] Ny EC rehetra manana fampianarana iray farafahakeliny
- [ ] Ny EC tsirairay manana quiz fanekena
- [ ] Ny video dia azo jerena
- [ ] Ny tahirin-kevitra misokatra tsara
- [ ] Ny filaharan'ny fampianarana dia lojika

### Toetry ny lesona

| Toetran | Famaritana |
|---------|------------|
| **Brouillon** | Hitanao irery |
| **Am-pandinihana** | Nalefa ho an'ny fanekena admin |
| **Navoaka** | Hitan'ny mpianatra voasoratra |
| **Voatahiry** | Afenina fa voatahiry |

### Mandefa ho an'ny famoahana

1. Tsindrio **Mandefa ho an'ny fandinihana**
2. Ny administration dia manamarina ny votoaty
3. Mandray mailaka fanekena na fanamarihana ianao
4. Rehefa nekena, ny lesona dia lasa **Navoaka**
