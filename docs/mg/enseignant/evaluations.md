# Fanombanana

Torolalana hamoronana sy hitantanana ny fanombanana amin'ny lesonao.

## Karazana fanombanana

| Karazana | Famaritana | Fanitsiana |
|----------|------------|------------|
| **Quiz EC** | QCM amin'ny faran'ny EC | Automatika |
| **Asa** | Asa omena | Amin'ny tanana |
| **Fanadinana farany** | Mivantana | Amin'ny tanana |

## Quiz automatika

### Mamorona quiz

1. Ao amin'ny lesonao, midira amin'ny EC voakasika
2. Tsindrio **Manampy quiz**
3. Amboary ny paramètres
4. Ampio ny fanontanianao

### Banky fanontaniana

Mamorona banky fanontaniana hanovana ny quiz:

```
Banky "Python Fototra" (20 fanontaniana)
├── Mora (8 fanontaniana) - Lanja 30%
├── Antonony (8 fanontaniana) - Lanja 50%
└── Sarotra (4 fanontaniana) - Lanja 20%

Quiz novokarina: 10 fanontaniana kisendrasendra araka ny lanja
```

### Paramètres mandroso

| Paramètre | Safidy | Voatolotra |
|-----------|--------|------------|
| **Mampifangaro fanontaniana** | Eny/Tsia | Eny |
| **Mampifangaro valiny** | Eny/Tsia | Eny |
| **Mampiseho isa** | Avy hatrany/Atao ela | Avy hatrany |
| **Mampiseho fanitsiana** | Eny/Tsia/Aorian'ny deadline | Aorian'ny deadline |

## Asa

### Mamorona asa

1. Ao amin'ny EC, tsindrio **Manampy asa**
2. Fenoy ny fampahalalana:

| Saha | Famaritana |
|------|------------|
| **Lohateny** | Anaran'ny asa |
| **Torolalana** | Toro-hevitra amin'ny antsipiriany |
| **Fe-potoana** | Deadline fanolorana |
| **Format ekena** | PDF, DOCX, ZIP, sns. |
| **Habe max** | Amin'ny Mo |
| **Barème** | Points na /20 |

### Ohatra torolalana

```markdown
## Asa: Projet Python

### Tanjona
Mamorona programa Python izay...

### Omena
1. Code source (.py)
2. Tatitra fanazavana (PDF, 2-3 pejy)
3. Capture écran ny valiny

### Critères d'évaluation
- Fiasan'ny code: 10 pts
- Kalitaon'ny code: 5 pts
- Tatitra: 5 pts

### Fe-potoana
15 novambra 2024, 23h59 (ora Madagasikara)
```

### Fitantanana ny rendus

Midira amin'ny **Ny Lesoko** > **[Lesona]** > **Asa** > **Jereo ny rendus**

## Manitsiana ny asa

### Interface fanitsiana

1. Tsindrio rendu iray
2. Alao/jereo ny rakitra
3. Ampiasao ny grille d'évaluation
4. Ampio fanamarihana
5. Omeo ny naoty
6. **Avoaka ny naoty**

### Grille d'évaluation

Mamorona grille hanaovana standardisation ny fanitsiana:

```
Critère                  | Tsara be | Tsara | Antonony | Tsy ampy
-------------------------|----------|-------|----------|----------
Code mandeha             | 10       | 7     | 4        | 0
Kalitaon'ny code         | 5        | 4     | 2        | 0
Documentation            | 5        | 4     | 2        | 0
```

### Fanamarihana manorina

::: tip Fomba tsara
- Manokàna: "Ny boucle ligne 15 dia azo amboarina" fa tsy "Code hatsaraina"
- Mampitovy ny tsara sy ny ratsy
- Manolo-kevitra fanatsarana
:::

## Fanaraha-maso ny valiny

### Tableau de bord fanombanana

Midira amin'ny **Ny Lesoko** > **Statistika** hijery:

- Moyenne ny kilasy isaky ny EC
- Fizarazaran'ny naoty
- Tahan'ny fahombiazana
- Mpianatra sahirana

### Mamoaka ny naoty

1. **Statistika** > **Mamoaka**
2. Safidio ny format: Excel, CSV, PDF
3. Safidio ny colonne hampidirina
4. Alao

## Fanadinana farany

### Fanomanana

Ny fanadinana farany dia karakarain'ny administration:

1. Alefao ny sujet-nao 3 herinandro mialoha
2. Manomàna ny barème amin'ny antsipiriany
3. Manomàna version rattrapage

### Format ny sujet

```
CNTEMAD - Fanadinana [Sampana] [Ambaratonga]
Session: [Daty]
Faharetana: [X] ora
Tahirin-kevitra: [azo atao/tsy azo atao]

Fizarana 1: QCM (30 points)
...

Fizarana 2: Fanazaran-tena (50 points)
...

Fizarana 3: Olana (20 points)
...
```

### Fanitsiana ny copie

1. Ny copie voascan dia misy ao amin'ny système
2. Tsindrio amin'ny interface an-tserasera
3. Na alao ho an'ny fanitsiana offline
4. Ampidiro ny naoty ao amin'ny système
5. Ny naoty dia ekena amin'ny administration

## Halatra sy hosoka

### Fikarohana automatika

Ny système dia manamarina:
- Fitoviana eo amin'ny copie
- Copie avy amin'ny internet (ho an'ny asa lahatsoratra)

### Raha misy halatra hita

1. Mandray alerte ianao
2. Manamarina amin'ny tanana
3. Milaza amin'ny administration raha voamarina
4. Ny administration no mandray fepetra ara-dalàna
