### Admin Nasionaly

Ny dashboard nasionaly dia manangona ny data avy amin'ny foibe 34:

| Widget | Famaritana |
|--------|------------|
| **Saritany interactive** | Fizarazaran'ny mpianatra isaky ny foibe |
| **KPI nasionaly** | Total mpianatra, vola miditra, taha fahombiazana |
| **Fampitahana foibe** | Laharana ny foibe araka ny fahombiazana |
| **Alerte** | Olana mila fiheverana |

## Widget misy

### Statistika mpianatra

- **Total mpianatra** : Mavitrika ao amin'ny foibe-nao
- **Vaovao ity volana** : Fisoratana anarana vao haingana
- **Isaky ny sampana** : Fizarazarana Informatika, LalÃ na, sns.
- **Isaky ny ambaratonga** : L1, L2, L3, M1, M2

### Statistika ara-bola

- **Vola miditra ny volana** : Total fandoavana voamarina
- **Fandoavana miandry** : Tokony homarinina amin'ny tanana raha ilaina
- **Fampitahana N-1** : Fivoarana vs taona lasa
- **Vinavina** : Estimation faran'ny volana

### Fandrosoana ara-pianarana

- **Taha fahombiazana** : % EC voamarina / total
- **Mpianatra sahirana** : <50% fandrosoana
- **Tsara indrindra** : Mpianatra mahomby indrindra

## Fitantanana mpianatra

### Mitady mpianatra

1. **Menu** > **Mpianatra** > **Mitady**
2. Tadiavo amin'ny: Anarana, mailaka, laharana mpianatra
3. Tsindrio valiny iray hijery ny profil

### Profil mpianatra

Ny profil dia mampiseho:
- Fampahalalana manokana
- Tantaran'ny fandoavana
- Fandrosoana amin'ny lesona
- EC voamarina / eo am-pandehanana

### Hetsika azo atao

| Hetsika | Famaritana |
|---------|------------|
| **Manova profil** | Manitsiana fampahalalana |
| **Mamerina tenimiafina** | Mandefa rohy reset |
| **Manatsahatra kaonty** | Manakana vonjimaika |
| **Mamoaka data** | PDF ny dossier mpianatra |
