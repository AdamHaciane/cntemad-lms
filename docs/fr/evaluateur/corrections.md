# Mes Corrections

Liste des soumissions en attente de correction.

## Accès

Connectez-vous avec votre compte Évaluateur.

## File d'attente

La liste affiche les soumissions à corriger :
- Nom de l'étudiant
- EC concerné
- Date de soumission
- Priorité

### Tri

Par défaut, les plus anciennes soumissions apparaissent en premier (FIFO).

Vous pouvez trier par :
- Date de soumission
- EC
- Priorité

### Filtres

- **EC** : Filtrer par élément constitutif
- **Date** : Période de soumission

## Actions

| Action | Description |
|--------|-------------|
| **Corriger** | Ouvrir la page de correction |
| **Reporter** | Différer la correction |

## Badge

Le nombre de corrections en attente s'affiche dans le menu.

## Bonnes pratiques

- Corrigez dans l'ordre d'arrivée
- Complétez au moins 10 corrections par jour
- Donnez des feedbacks constructifs
