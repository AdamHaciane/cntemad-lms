# Validation des Certificats

Approuvez les certificats des étudiants ayant complété leur année.

## Éligibilité

Un étudiant est éligible au certificat quand :
- Tous les EC de l'année sont validés
- Toutes les évaluations sont passées
- Aucun impayé en cours

## Liste des éligibles

La page affiche les étudiants en attente de certification :
- Nom de l'étudiant
- Année (L1, L2, L3, M1, M2)
- Centre
- Date d'éligibilité

## Vérification

Avant d'approuver, vérifiez :
1. La progression complète de l'étudiant
2. Les scores obtenus
3. L'absence d'anomalies

## Actions

### Approuver

1. Cliquez sur **Approuver**
2. Le certificat PDF est généré automatiquement
3. L'étudiant est notifié par email/SMS

### Rejeter

1. Cliquez sur **Rejeter**
2. Saisissez le motif
3. L'étudiant est notifié avec le motif

## Certificat généré

Le certificat inclut :
- Logo CNTEMAD
- Nom complet de l'étudiant
- Année validée
- Date d'émission
- Numéro unique de vérification
- Signature numérique

## Vérification externe

Les certificats peuvent être vérifiés sur :
`https://verify.cntemad.mg/[numero]`
