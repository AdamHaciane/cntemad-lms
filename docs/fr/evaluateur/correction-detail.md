# Correction Détaillée

Comment corriger une soumission d'étudiant.

## Interface

La page de correction affiche :
- À gauche : La soumission de l'étudiant
- À droite : Vos outils de correction

## Étapes de correction

### 1. Lire la soumission

Lisez attentivement le travail de l'étudiant.

### 2. Attribuer une note

Utilisez le composant de note :
- Échelle de 0 à 20
- Boutons +/- pour ajuster
- La mention s'affiche automatiquement

| Note | Mention |
|------|---------|
| 16-20 | Très Bien |
| 14-15.99 | Bien |
| 12-13.99 | Assez Bien |
| 10-11.99 | Passable |
| 0-9.99 | Insuffisant |

### 3. Rédiger un feedback

Donnez un retour constructif :
- Points forts
- Axes d'amélioration
- Conseils pour progresser

Utilisez les templates prédéfinis si besoin.

### 4. Valider ou Rejeter

- **Valider** : La note est enregistrée, l'étudiant est notifié
- **Rejeter** : Travail refusé, l'étudiant doit resoumettre
- **Sauvegarder brouillon** : Continuer plus tard

## Navigation automatique

Après validation, vous êtes redirigé vers la soumission suivante.

## Conseils

- Soyez précis dans vos feedbacks
- Restez encourageant même pour les notes basses
- Signalez les cas de plagiat à l'administration
