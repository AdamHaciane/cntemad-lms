# Paramètres de Notifications

Configurez comment vous souhaitez être informé.

## Types de notifications

| Type | Description |
|------|-------------|
| **EC validé** | Votre enfant a validé un EC |
| **Paiement reçu** | Confirmation de paiement |
| **Inactivité** | Enfant inactif depuis 7+ jours |
| **Quiz passé** | Résultat d'un quiz |
| **Certificat** | Certificat disponible |

## Canaux de notification

### Email

- Activé par défaut
- Adresse : celle de votre compte
- Modifier dans Profil > Email

### SMS

- Optionnel
- Numéro : celui de votre compte
- Coût : gratuit (envoyé par CNTEMAD)

### Application

- Notifications push (si app mobile)
- Badge dans le menu web

## Fréquence

### Notifications immédiates

- EC validé
- Paiement

### Récapitulatif hebdomadaire

Email chaque lundi avec :
- Progression de la semaine
- EC validés
- Activité générale

Désactivable dans les paramètres.

## Désactiver des notifications

1. Allez dans **Paramètres**
2. Section **Notifications**
3. Décochez les types non souhaités
4. Sauvegardez

## Attention

Certaines notifications critiques ne peuvent pas être désactivées :
- Problèmes de sécurité
- Changements importants du compte
