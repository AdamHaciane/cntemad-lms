# Paiements pour mon Enfant

Payez les EC de votre enfant.

## Historique des paiements

Liste des paiements effectués :
- Date
- EC concerné
- Montant
- Provider utilisé
- Statut

## Payer un EC

### Étape 1 : Sélectionner l'enfant

Si vous avez plusieurs enfants liés, sélectionnez celui concerné.

### Étape 2 : Choisir l'EC

Liste des EC non payés avec :
- Titre de l'EC
- Prix
- Description

### Étape 3 : Paiement

1. Sélectionnez le provider :
   - MVola (Telma)
   - Orange Money
   - Airtel Money

2. Saisissez votre numéro de téléphone

3. Cliquez sur **Payer**

4. Confirmez sur votre téléphone

### Étape 4 : Confirmation

Vous recevez :
- SMS de confirmation
- Reçu par email
- Notification dans l'app

Le reçu est au nom du parent (vous).

## Reçus

Téléchargez vos reçus pour :

- Justificatifs fiscaux
