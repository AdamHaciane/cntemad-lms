# Mes Enfants

Suivez les études de vos enfants au CNTEMAD.

## Accès

Connectez-vous avec votre compte Parent/Tuteur.

## Vue d'ensemble

Chaque enfant est affiché dans une carte avec :
- Photo et nom
- Année en cours (L1, L2, L3, M1, M2)
- Progression globale
- Dernière activité

## Ajouter un enfant

Pour lier un compte étudiant :
1. Demandez le code de liaison à votre enfant
2. Cliquez sur **Ajouter un enfant**
3. Saisissez le code
4. Validez

L'étudiant doit approuver la liaison depuis son compte.

## Actions rapides

| Action | Description |
|--------|-------------|
| **Voir progression** | Détail des EC |
| **Payer** | Payer un EC pour l'enfant |
| **Contacter** | Envoyer un message |

## Alertes

Vous êtes alerté quand :
- Votre enfant n'a pas été actif depuis 7 jours
- Un paiement est en attente
- Votre enfant valide un EC
