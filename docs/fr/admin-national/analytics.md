# Analytics National

Analysez les données de l'ensemble du réseau CNTEMAD.

## Tableaux de bord

### Tendances d'inscription

- Évolution mensuelle des inscriptions
- Comparaison année sur année
- Prévisions

### Revenus

- Total par mois
- Répartition par provider (MVola, Orange, Airtel)
- Croissance

### Performance académique

- Taux de validation par niveau
- Scores moyens aux quiz
- Certificats délivrés

## Graphiques disponibles

- **Courbes** : Évolution temporelle
- **Barres** : Comparaison entre centres
- **Camembert** : Répartition
- **Heatmap** : Activité par jour/heure

## Filtres

- Période : Jour, Semaine, Mois, Année
- Centre(s) : Un ou plusieurs
- Niveau : L1 à M2
- Provider : MVola, Orange, Airtel

## Export

Exportez les données pour analyse externe :
- Format CSV
- Format Excel
- Format PDF (avec graphiques)

## Rapports automatiques

Configurez des rapports automatiques :
- Quotidiens (KPIs clés)
- Hebdomadaires (résumé)
- Mensuels (rapport complet)

Envoi par email aux destinataires configurés.
