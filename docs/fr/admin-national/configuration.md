# Configuration Système

Paramètres globaux de la plateforme CNTEMAD LMS.

## Paramètres généraux

### Année universitaire

- Définir les dates de l'année en cours
- Configurer les périodes d'inscription
- Gérer les vacances

### Paiements

- Activer/désactiver les providers
- Configurer les montants par EC
- Définir les délais de validation

### Notifications

- Templates email
- Templates SMS
- Fréquence des rappels

## Gestion des utilisateurs

### Rôles

| Rôle | Permissions |
|------|-------------|
| Étudiant | Voir cours, payer, quiz |
| Enseignant | Créer contenu, voir stats |
| Évaluateur | Corriger, valider certificats |
| Mentor | Suivre mentorés, messagerie |
| Admin Centre | Gérer son centre |
| Admin National | Tout accès |

### Utilisateurs

- Créer des comptes admin
- Réinitialiser des mots de passe
- Suspendre des comptes
