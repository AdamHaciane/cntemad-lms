# Gestion des Centres

Gérez les 34 centres régionaux CNTEMAD.

## Liste des centres

Vue tabulaire de tous les centres avec :
- Nom du centre
- Région
- Responsable
- Nombre d'étudiants
- Revenus du mois
- Statut

## Détail d'un centre

Cliquez sur un centre pour voir :
- Informations générales
- Équipe administrative
- KPIs détaillés
- Historique d'activité

## Comparaison de centres

Comparez les performances de plusieurs centres :

1. Sélectionnez 2 à 5 centres
2. Cliquez sur **Comparer**
3. Visualisez les graphiques comparatifs

### Métriques comparées

- Inscriptions par mois
- Revenus
- Taux de validation
- Satisfaction étudiants

## Actions

| Action | Description |
|--------|-------------|
| **Exporter** | Données d'un ou plusieurs centres |
| **Contacter** | Envoyer message au responsable |
| **Suspendre** | Suspendre temporairement un centre |

## Ajout d'un centre

Pour ajouter un nouveau centre :
1. Cliquez sur **Nouveau centre**
2. Renseignez les informations
3. Assignez un responsable
4. Validez

## Support

En cas de problème avec un centre, contactez le support technique.
