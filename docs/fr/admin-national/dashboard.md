# Dashboard National

Vue globale des 34 centres CNTEMAD de Madagascar.

## Accès

Réservé aux administrateurs nationaux CNTEMAD.

## KPIs nationaux

| Indicateur | Description |
|------------|-------------|
| **Total étudiants** | Somme des 34 centres |
| **Paiements totaux** | Revenus du mois |
| **Taux moyen validation** | Moyenne nationale |
| **Centres actifs** | Centres avec activité récente |

## Carte Madagascar

Visualisez les 34 centres sur une carte interactive :
- **Vert** : Centre performant
- **Jaune** : Performance moyenne
- **Rouge** : Attention requise

Cliquez sur un centre pour voir ses détails.

## Top 5 centres

Liste des centres avec les meilleures performances :
- Nombre d'inscriptions
- Revenus générés
- Taux de validation

## Alertes

- Centres sans activité depuis 7+ jours
- Paiements en anomalie
- Problèmes techniques signalés
