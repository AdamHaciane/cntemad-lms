# Inscription

Bienvenue au CNTEMAD ! Ce guide vous accompagne pour créer votre compte étudiant.

## Prérequis

- Un numéro de téléphone valide (034, 032, 033, 038)
- Une adresse email
- Votre numéro CIN ou passeport

## Étapes d'inscription

### 1. Accéder au formulaire

Rendez-vous sur la page d'inscription : [lms.cntemad.mg/inscription](https://lms.cntemad.mg/inscription)

### 2. Remplir vos informations

| Champ | Description |
|-------|-------------|
| **Nom & Prénom** | Tels qu'indiqués sur votre CIN |
| **Email** | Adresse email valide |
| **Téléphone** | Numéro mobile (pour paiement) |
| **Centre régional** | Choisissez votre centre le plus proche |
| **Filière** | Informatique, Droit, Économie, etc. |
| **Niveau** | L1, L2, L3, M1, M2 |

### 3. Vérification

Un code de vérification sera envoyé par SMS à votre numéro.

```
Exemple de SMS reçu :
"CNTEMAD - Votre code de vérification : 123456"
```

### 4. Confirmation

Une fois le code validé, votre compte est créé. Vous recevrez un email de confirmation.

## Choisir vos EC (Éléments Constitutifs)

Après inscription, vous devez sélectionner vos EC pour l'année universitaire :

1. Accédez à **Mon Parcours** > **Choisir mes EC**
2. Sélectionnez les EC selon votre niveau et filière
3. Confirmez votre sélection
4. Procédez au [paiement](/fr/etudiant/paiement)

::: tip Conseil
Commencez par les EC obligatoires de votre niveau avant de choisir des EC optionnels.
:::

## Problèmes fréquents

### Je ne reçois pas le code SMS

- Vérifiez que votre numéro est correct
- Attendez 2 minutes avant de redemander
- Vérifiez que vous avez du crédit téléphonique

### Mon email est déjà utilisé

Vous avez peut-être déjà un compte. Utilisez la fonction **Mot de passe oublié** pour récupérer l'accès.

### Je me suis trompé de centre régional

Contactez votre centre régional pour modifier cette information après inscription.
