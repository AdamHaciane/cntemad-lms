# Dashboard Centre Régional

Le dashboard centre vous donne une vue d'ensemble de l'activité de votre centre CNTEMAD.

## Accès

1. Connectez-vous avec votre compte Admin Centre
2. Vous arrivez directement sur le dashboard

## KPIs affichés

| Indicateur | Description |
|------------|-------------|
| **Étudiants inscrits** | Nombre total d'étudiants de votre centre |
| **Étudiants actifs** | Étudiants ayant une activité ce mois |
| **Paiements du mois** | Nombre et montant total |
| **Taux de validation** | % EC validés par rapport aux payés |

## Actions rapides

- **Ajouter un étudiant** : Inscription manuelle
- **Voir les paiements** : Accès à la liste des paiements
- **Générer un rapport** : Export mensuel

## Alertes

Le dashboard affiche les alertes importantes :
- Paiements en attente de validation
- Étudiants inactifs depuis plus de 7 jours
