# Gestion des Paiements

Gérez les paiements des étudiants de votre centre.

## Vue d'ensemble

La page affiche tous les paiements avec :
- Nom de l'étudiant
- EC concerné
- Montant
- Provider (MVola, Orange, Airtel)
- Statut
- Date

## Statuts de paiement

| Statut | Description |
|--------|-------------|
| **En attente** | Paiement initié, en attente de confirmation |
| **En traitement** | Confirmation en cours auprès du provider |
| **Réussi** | Paiement confirmé, accès accordé |
| **Échoué** | Paiement refusé ou annulé |

## Actions

### Validation manuelle

En cas de problème avec la confirmation automatique :
1. Vérifiez avec l'étudiant la preuve de paiement
2. Cliquez sur **Valider manuellement**
3. Confirmez l'action

### Remboursement

Pour les cas exceptionnels :
1. Sélectionnez le paiement
2. Cliquez sur **Rembourser**
3. Saisissez le motif
4. Le remboursement sera traité sous 48h

## Filtres

- **Statut** : En attente, Réussi, Échoué
- **Provider** : MVola, Orange, Airtel
- **Période** : Dates de début et fin

## Export

Exportez les paiements en CSV pour votre comptabilité.
