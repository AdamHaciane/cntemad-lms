# Gestion des Étudiants

Cette page vous permet de gérer les étudiants inscrits dans votre centre.

## Fonctionnalités

### Recherche et filtres

- **Recherche** : Par nom, email ou numéro d'inscription
- **Filtres** :
  - Année (L1, L2, L3, M1, M2)
  - Statut (Actif, Inactif, Suspendu, Diplômé)
  - Période d'inscription

### Actions disponibles

| Action | Description |
|--------|-------------|
| **Voir détail** | Accéder au profil complet de l'étudiant |
| **Suspendre** | Suspendre temporairement un compte |
| **Exporter** | Télécharger la liste en CSV |

## Export des données

Cliquez sur **Exporter** pour télécharger :
- Liste complète des étudiants
- Avec filtres appliqués
- Format CSV compatible Excel

## Bonnes pratiques

- Vérifiez régulièrement les étudiants inactifs
- Contactez les étudiants suspendus pour régularisation
- Mettez à jour les informations de contact
