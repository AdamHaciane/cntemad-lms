# Rapports Centre

Générez des rapports détaillés pour votre centre.

## Types de rapports

### Rapport mensuel

- Nombre d'inscriptions
- Paiements reçus (montant total)
- Taux de validation des EC
- Étudiants actifs vs inactifs

### Rapport par année universitaire

- Progression par niveau (L1, L2, L3, M1, M2)
- Taux de réussite aux quiz
- Certificats délivrés

### Rapport financier

- Détail des paiements par provider
- Réconciliation comptable
- Montants en attente

## Générer un rapport

1. Sélectionnez le type de rapport
2. Choisissez la période
3. Cliquez sur **Générer**
4. Téléchargez en PDF ou CSV

## Planification automatique

Configurez l'envoi automatique de rapports :
- Fréquence : Hebdomadaire, Mensuel
- Destinataires : Email(s)
- Format : PDF, CSV

## Bonnes pratiques

- Générez un rapport mensuel en fin de mois
- Partagez les KPIs avec l'administration centrale
- Archivez les rapports pour l'historique
